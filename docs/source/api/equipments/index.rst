Equipments
==========

Equipment objects represent physical components (e.g., membranes, electrodes, columns) that can be attached to :class:`~qsdsan.SanUnit` objects to track construction inventories, costs, and material inputs for TEA and LCA.

All classes are also importable directly from :mod:`qsdsan.equipments`.
